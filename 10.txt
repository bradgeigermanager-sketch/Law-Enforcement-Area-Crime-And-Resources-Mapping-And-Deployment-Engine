## 3. Core Engine Mechanics

* Webhook Threat Alerting System (Tab 1): Scans downstream model outputs to identify rapid growth anomalies. Areas where the spike exceeds defined parameters generate standardized JSON notification payloads, ready to route to dispatch center endpoints.
* Linear Simplex Optimization Optimizer (Tab 2): Replaces guesswork with resource optimization. It uses bounded simplex linear programming via scipy.optimize.linprog to maximize coverage across high-risk locations. The allocation algorithm is governed by explicit system resource constraints:
$$\sum \text{Units} \le \text{Total Staff}$$ 
* Data Compilation Engine (Tab 3): Formats calculations, historical parameters, and deployment metrics into an unformatted byte buffer. This enables instant text downloading for record-keeping and auditable performance reviews.
